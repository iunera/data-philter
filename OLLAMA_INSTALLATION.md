# Ollama Installation Guide

This guide provides instructions on how to install Ollama on various operating systems.

## macOS

1.  Download the Ollama application from the [official website](https://ollama.com).
2.  Open the downloaded file and drag the Ollama application to your Applications folder.
3.  Launch the Ollama application from your Applications folder.

## Windows

1.  Download the Ollama installer from the [official website](https://ollama.com).
2.  Run the installer and follow the on-screen instructions.
3.  Once the installation is complete, Ollama will be running in the background.

## Linux

You can install Ollama on Linux using a single command:

```bash
curl -fsSL https://ollama.com/install.sh | sh
```

This command will download and run the installation script.

For more detailed instructions and troubleshooting, please refer to the [official Ollama documentation](https://github.com/ollama/ollama/blob/main/docs/linux.md).
